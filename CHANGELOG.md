# Changelog

## V4 (4.0)

ITAM V4 ships a faster, more visual admin experience while keeping the same Django + Supabase core.

### Highlights

- In-page **Add Asset** and **Add Employee** modals (API-backed creates with notifications)
- Assets **action dock** (Add / Import / Export) and redesigned **filter bar**
- Dashboard welcome **day/night sky** and analytics polish
- Version surface area updated to **4.0** (`inventory/version.py`, package metadata, README)

### Source of truth

- App version: `inventory/version.py` → `ITAM_VERSION = "4.0"`
- Package version: `pyproject.toml` → `4.0.0`
